# OAuth2 Slides

> revealjs slideshow presentation about OAuth2 and OpenID Connect

## Installation

Install dependencies:

```ruby
npm install
```

## Development

Generate slideshow HTML:

```ruby
npm run build
```

Start development server with livereload enabled:

```ruby
npm run start
```
